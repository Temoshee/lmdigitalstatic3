How to Search this Collection

This website is useful to researchers who already have a public act number and year in hand. Subject matter and MCL number searches will not produce any results.

To find documents related to a specific public act, enter the year and public act number into the above search box using the following format 1976-PA-390. The same type of search can be performed for bill numbers. For example: 1995-SB-344.

This website does not search the text of any documents. It only looks for public act numbers and bill numbers.
